# Affect Counsel Unity

[한국어](README.ko.md) · [English](README.en.md)

한국어 1:1 상담사 훈련을 위한 affect-aware Unity 시뮬레이션입니다. 상담자의 언어적 미세기술과 얼굴·시선·움직임 같은 비언어적 전달이 내담자의 관계 안전감과 공개 의지에 어떻게 함께 작용하는지 연습합니다.

An affect-aware Unity simulation for Korean one-to-one counselor training. It helps trainees examine how counseling micro-skills and embodied delivery jointly influence relational safety and client disclosure.
